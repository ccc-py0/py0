# Script1 language: system library

This package is the system library for script1 language

## Features

1. console
2. array
3. base
4. datetime
5. fs
6. hash
7. json
8. math
9. regexp
10. str
11. type

## Getting started

```dart
import 'package:script1/script1.dart';
```

import 'package:script1/script1.dart';

  log('hello');
